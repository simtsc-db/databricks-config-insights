## How to capture / refresh them

1. Deploy the bundle and run the collector at least twice (drift needs two runs).
2. Open the **Configuration Insights** dashboard in your workspace.
3. For each page, take a screenshot (macOS: `Cmd+Shift+4`, then space to capture a
   window) and save it here using the file names above.
4. Commit the images. They render automatically in the top-level `README.md`.

Keep images reasonably sized (≈ 1600px wide, PNG) so the README stays lightweight.
